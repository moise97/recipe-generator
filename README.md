# AI Recipe Generator 🍳

Turn whatever's in your fridge into delicious recipes — powered by Google Gemini AI.

![Recipe Generator Demo](demo/demo.gif)

## Live Demo

👉 [Try it live on Hugging Face](https://huggingface.co/spaces/moise97/recipe-generator)

## What it does

- Enter any ingredients you have at home
- Get 3 instant recipe suggestions with full instructions
- Supports dietary preferences: Vegetarian, Vegan, Gluten-free, Low-carb
- Shows cook time, difficulty, and nutrition info for each recipe
- Upload a photo of your fridge and let AI identify your ingredients

## Tech stack

- **Python 3.12** — core language
- **Google Gemini AI** — recipe generation via prompt engineering
- **Gradio** — web interface
- **Hugging Face Spaces** — free live deployment
